Examples Of UI Web Apps Using Vuejs, Vuetifyjs, Vanilla JavaScript And Axios


Overlapping Studies UI

My first project using Vuejs, Vuetifyjs, and Axios. It allows the user to enter unique values from a data table and it sents it to a protocol where it issues a unique sessionID which the UI captures it and then sends it to the awaiting dashboard which loads up the panel with only that particular information saving loading time. In my workplace, there was a frustration of loading dashboards from both the user and the data scientist. This method of isolating only the particular measures proved useful as some panels took up to 20 minutes to load because of the amount of data that is initially loading onto the boards.

BBDash UI

In this scenario, I added a typeahead search bar that once the users make a choice, it makes it into a chip. I also added a large modal where there is an extended search of up to 10 filters where the user enters the criteria. Once they click on submit. It does the same thing as Overlapping studies, it generates a session id with those specific values, and it sends the session id to the dashboard to load up the panel with only those specific values. This dashboard took up to 15 mins just to load up. Now it takes less than a minutes.

Clinical Samples UI

This UI is built with the same intentions as the other two UIs; to minimize the loading time by only calling the necessary values the user needs at the time. The only difference from this one is that I used a cascading endpoint to effectively call up the right combinations of values that would render a successful query.
